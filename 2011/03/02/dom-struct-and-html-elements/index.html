DOM结构以及HTML中的基本元素

Mar 2, 2011

DOM的基本结构：

几点说明:
1.DOM的根节点是Document文档，每个载入浏览器的 HTML 文档都会成为 Document 对象,window.document代表了这个对象，也可以用所有基本元素节点的ownerDocument属性获取这个对象。
2.html节点是第二级节点，包含了head和body两个元素节点。
