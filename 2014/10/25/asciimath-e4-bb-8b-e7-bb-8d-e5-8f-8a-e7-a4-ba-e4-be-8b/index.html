AsciiMath介绍及示例

Oct 25, 2014

在网页显示数学公式并不是十分容易，其实在其他地方比如word中插入公式也不是容易的事。

直到最近几年随着HTML5的普及和浏览器对MathML标准的支持，以及强大的MathJax( JavaScript display engine for LaTeX, MathML, and AsciiMath)的出现，才让数学公式在网页上书写和显示成为可能(使用图片不在讨论范围)。

但是使用LaTeX语法或MathML语法书写数学公式依然很复杂，查普曼大学数学系计算科学学院的Peter Jipsen教授为了和学生在网络交流方便发明了AsciiMath语法。

下面我们来对比一下用各种语法书写 ( x = \frac {-b\pm \sqrt {b^{2}-4ac}} {2a} ) 这个公式：

[mathjax]
LaTeX语法 

 x = \frac {-b\pm \sqrt {b^{2}-4ac}} {2a}

MathML语法
<math xmlns=”http://www.w3.org/1998/Math/MathML"&gt;
  <mstyle displaystyle=”true”>
    <mi> x </mi>
    <mo> = </mo>
    <mfrac>
      <mrow>
        <mo> - </mo>
        <mi> b </mi>
        <mo> ±<!–plus-minus sign–> </mo>
        <msqrt>
          <msup>
            <mrow>
              <mi> b </mi>
            </mrow>
            <mrow>
              <mn> 2 </mn>
            </mrow>
          </msup>
          <mo> - </mo>
          <mn> 4 </mn>
          <mi> a </mi>
          <mi> c </mi>
        </msqrt>
      </mrow>
      <mrow>
        <mn> 2 </mn>
        <mi> a </mi>
      </mrow>
    </mfrac>
  </mstyle>
</math>

AsciiMath语法

 x = (-b +- sqrt(b^2-4ac))/(2a)

以上对比可以看到AsciiMath语法最简洁直观，和高级计算器的输入显示格式类似。
